## 📖 About

**Escape from Dungeon** is a top-down RPG / dungeon crawler where you must fight your way through a dark dungeon and defeat the boss to unlock a teleportation gate — your only way out.

---

## 🎮 Gameplay

- Explore a dungeon filled with enemies standing between you and freedom.
- **Defeat the boss** to reveal the teleportation gate.
- Use the gate to escape and complete the game.

---

## ✨ Features

- ⚔️ **Combat System** — Attack enemies using melee or ranged weapons with mouse input.
- 🎒 **Inventory System** — Manage your equipped weapons through inventory slots.
- 🏃 **Smooth Movement** — Responsive top-down character movement.
- ✨ **Custom Shaders** — Visual effects powered by ShaderLab & HLSL for atmospheric dungeon aesthetics.

## 🚀 Getting Started

### Prerequisites

- [Unity Hub](https://unity.com/download) with Unity **2021.x** or later installed.

### Run the Project

1. Clone the repository:
```bash
   git clone https://github.com/NgoVuongQuy/Escape-from-Dungeon.git
```
2. Open **Unity Hub** → click **Open** → select the cloned folder.
3. Wait for Unity to import all assets.
4. Open the main scene in `Assets/Scenes/` and press **Play**.
